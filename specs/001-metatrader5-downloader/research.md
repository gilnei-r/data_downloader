# Research: Metatrader5 Data Downloader

## Performance Goals

**Decision**: The application should be able to download data for 1000 tickers in under 5 minutes.
**Rationale**: This is a reasonable starting point for performance, and can be optimized later if needed.
**Alternatives considered**: None.

## Scale/Scope

**Decision**: The application should be able to handle up to 5000 tickers in a single run.
**Rationale**: This provides a good balance between performance and resource usage.
**Alternatives considered**: None.
